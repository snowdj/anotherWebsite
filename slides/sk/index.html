Assumption of the Model

  1. Cost is fixed

    • For cerntain area and time period, we assume it follows a quadratic equation. \[C_{ij} = a X_{ij} + b X_{ij}^2\]
    • i is time; j is crop number

  2. Yield is normally distributed (Turvey, 2012)

    • Yield depends on random factors like weather. \[Y_{ij} \sim N(E[Y_j], \sigma(Y_j))\]

  3. Price followS a stochastic process.

    • Price changes frequently, like other commodity price. \[P_{ij} = P_{0j} e^{((\mu - \frac{1}{2}) \frac{7}{12} + \sigma N(0,1) \sqrt{\frac{7}{12}})}\]

The Cost of Production: PMP Approach

  • Calibration method: Postive Mathematical Programming

  • Advantage:

    • Minimal data requirement
    • Calibrate MP models exactly to observed behaviour
    • Optimum: combination of binding constraints and first-order conditions
    • Prediction of consequences and sensitivity analysis

  • Three stages ( formalized by Howitt (1995a) )

    • Estimate output marginal cost
    • Estimate cost function
    • Check the model

Estimate Output Marginal Cost

  • Maximize farmer's net revenue including a set of calibration constraints.

\[Max: R = \sum_{j=1}^n ( p_j x_j y_j - c_j x_j)\] \[Subjet \; to: \sum_{j=1}^n x_j \le 1668 \;\; (1)\] \[x_j \ge 0\] \[x_j \le x_j^{obs} + 0.01,\; \forall j; \; \; [\lambda_j] \;\; (2)\]

  • \(p_j\) is price, \(x_j\) is land use, \(y_j\) is yield, \(c_j\) is cost, n = 5;
  • Where \((1)\) is nature resource constraint (land 1668);
  • \((2)\) is the calibration constraint. Solve the problem in GAMS \(\to\) the associated shadow price \(\lambda_j\) for each crop.

Estimate cost function

  • Assumption: a quadratic cost function: \(c_j = a x_j + b x_j^2\) for SK.
    \[\hat b_j = 2 \times \frac{\lambda_j}{x_j^{obs}} \; \text{and} \; \hat a_j=c_j^{obs} - 0.5 \times \hat b_j \times x_j^{obs} \]

    • Run a regression of yield on time using a polynomial form
    • Then the predict yield \(\hat Y_i\) is time trend
    • The standard deviation \(\sigma\) of residual \(\epsilon\) is what we want

Price Simulation: Geometric Brownian Model

  • Recall \[P_{ij} = P_{0j} e^{((\mu - \frac{1}{2}) \frac{7}{12} + \sigma N(0,1) \sqrt{\frac{7}{12}})}\]

    • \(P_{0j}\) is the initial spring price as of March 2014;
    • Price generation is based on a 7-month growing season;
      • \(P_{ij}\) is the random commodity harvest price generated by a log normal (Brownian) process with:
      • “drift”“ \(\mu\),
      • "volatility”“ \(\sigma\),
      • \(N(0,1)\) random deviation normal distributed;

Price Simulation: Geometric Brownian Model(Cont.)

  • Geometric Brownian Model follow this stochastic differential equation: \[dS_t = \mu S_t dt + \sigma S_t dW_t\]

    • \(\mu S_t dt\) is deterministic part; \(\mu S_t dW_t\) is stochastic part.
    • \(dW_t\) is the Brownian motion, which follows random normal distribution \(N(0,t)\).
    • \(\mu\) is drift; \(\sigma\) is diffusion. \(\sigma\) increases the amount of randomness entering the system.

Conclusion

  • With the cost, yield, and price data, the SK farm model can be structured as follows:

\[Min: \sigma_p^2 = \frac{1}{m} \sum_{i=1}^m ( \pi_i - E[\pi] )^2\] \[Subjet \; to: \sum_{j=1}^n x_j = 1668 \; \] \[\sum_{j=1}^n E[R_j] x_j = K \; (2)\]

  • \(\sigma_p^2\) measures the variance of farmers' profit ; \(m\) is 1000 simulation state
  • \(\pi_i\) is the profit for one state; we are going to count 1000 times
  • (2) \(K\) is the target profit
  • \(E[R]\) is expect the average net revenue per acre for each kind crop

Conclusion

  • The impact of policy can be added to model. For example,whole farm insurance:

\[\sum_{j=1}^n R_{1,j} x_j + Max[Z - \sum_{j=1}^n R_{1,j} x_j, \; \;0]- \frac{\delta}{m} \sum_{i=1}^m Max[Z - \sum_{j=1}^n R_{i,j} x_j, \;\; 0] - \pi_1 = 0 \] \[\sum_{j=1}^n R_{m,j} x_j + Max[Z - \sum_{j=1}^n R_{m,j} x_j, \;\; 0]- \frac{\delta}{m} \sum_{i=1}^m Max[Z - \sum_{j=1}^n R_{i,j} x_j, \;\; 0] - \pi_m = 0 \]

  • \(Z\) is the income coverage level to be protected by insurance for example 70% farmers history income
  • \(Max[Z - \sum_{j=1}^n R_{1,j} x_j, \;\; 0]\) is indemnity payout that farmer can get from insurance
  • \(\frac{1}{m} \sum_{i=1}^m Max[Z - \sum_{j=1}^n R_{i,j} x_j, \; 0]\) is permium, and farmers only need to pay \(\delta\) of it.
  • \(\delta\) is subsidy rate from government. If \(\delta\) = 0.50, the premium is subsidized by 50 percent.
